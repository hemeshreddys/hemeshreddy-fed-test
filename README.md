# hemeshreddy-fed-test


## Installation

Clone the project -> https://github.com/hemeshreddys/hemeshreddy-fed-test.git

Navigate to hemeshreddy-fed-test folder

install all the dependency
```bash
npm install
```
Run the project 
```bash
npm run dev
```
Run unit testing
```bash
npm run test
```
